## Enc2Btn AVRGCC Eclipse project  ##

Software is created using Eclipse Mars with AVR plugin 2.4.1 and AVR Toolchain 3.4.
Compiled intel hex file is provided in the /hex directory.

### ATTINY25 FUSE SETTINNGS ###
Low  = 0xE2
High = 0xDD
Ext  = 0xFF
